## 🚀 Getting Started & Installation

To run this project locally on your machine, follow these steps:

### 1. Clone the Repository
```bash
git clone [https://github.com/myintmyatmon15/myintmyatmon-portfolio.git](https://github.com/myintmyatmon15/myintmyatmon-portfolio.git)
cd myintmyatmon-portfolio
```
2. Install Dependencies (If using React/Node.js, skip if it's pure HTML/CSS)
```bash
npm install
```
3. Run the Local Development Server
```bash
# For React/Next.js apps
npm run dev # or npm start

# For Vanilla HTML/CSS
# Simply open the `index.html` file in your browser, or use VS Code "Live Server" extension.
```
