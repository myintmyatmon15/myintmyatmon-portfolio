# 📂 Myint Myat Mon — Portfolio Website

Welcome to the repository for my personal portfolio website! This project serves as a digital space to showcase my skills, professional experience, projects, and achievements as a Software Engineer specializing in Full Stack, Frontend Development, and QA Automation.

🌐 **Live Demo:** [https://myintmyatmon-portfolio.netlify.app/](https://myintmyatmon-portfolio.netlify.app/)

---

## ✨ Features

* **Responsive Design:** Seamlessly optimized for all screen sizes (Mobile, Tablet, and Desktop) ensuring a mobile-first flawless user experience.
* **Dynamic Projects Section:** Displays featured real-world projects (like Lumina Mon and CosmoFlix) with detailed descriptions, tech stacks, tags, and production deployment links.
* **Interactive Experience/Skills Timeline:** An engaging and structured way for recruiters to browse through my dual-expertise in Web Development and Quality Assurance.
* **Contact Form:** A secure and direct way for recruiters, engineering managers, and clients to get in touch with me immediately.
* **Modern UI/UX:** Built with smooth component transitions, intuitive navigation, and a professional, clean user aesthetic.

---

## 🛠️ Tech Stack

This website is built using modern, fast-loading, and scalable web technologies:

### Frontend Capabilities
* **Core Stack:** React 18, Next.js (App Router), TypeScript, JavaScript (ES6+)
* **Styling & UI:** Tailwind CSS, Material UI, Responsive Web Architecture
* **State Management:** Redux Toolkit, React Context API

### Quality Assurance & Systems
* **Testing & Scripts:** Unit Testing, Manual/Automated Test Scripting, Logic Debugging
* **Methodologies:** Agile/Scrum, Software Development Lifecycle (SDLC), System Validation (SIT/UAT)

### Tools & Deployment
* **Version Control:** Git, GitHub
* **Hosting & CI/CD:** Netlify, Vercel
